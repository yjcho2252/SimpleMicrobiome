# Release 2026-05-08

## Summary
- Decoupled Taxa Comparison regression-line display from significance-only taxa filtering.
- Updated Taxa Profiles manual wording to reflect q-value-based filtering and independent regression-line behavior.

## Changed
- Taxa Comparison (`modules/mod_taxa_comparison.R`):
  - Removed trend-mode-specific recalculation path from taxa filtering when `Show only q-value < 0.05` is enabled.
  - Significance-only taxa filtering now consistently uses taxa-level `q < 0.05` from `taxa_stats()`, regardless of `Show Regression Line`.
- Manual (`docs/manual/taxa-profiles.md`):
  - Updated trend annotation wording from `p-value` to `q-value`.
  - Removed outdated `Auto` description from `Group variable type`.
  - Clarified that `Show only q-value < 0.05` is independent of `Show Regression Line`.
