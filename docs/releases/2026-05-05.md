# Release 2026-05-05

## Summary
- Expanded taxonomy-rank handling to include `Strain` across key analysis modules.
- Improved MetaPhlAn/HUMAnN input conversion support through `scripts/generate_SM_format.py` output refinements.
- Updated Alpha Diversity rarefaction behavior to cap rarefaction depth at `100000` with explicit UI messaging.
- Added Random Forest package citation entry.

## Changed
- Alpha Diversity (`modules/mod_alpha.R`):
  - Rarefaction depth now uses `min(min(sample_sums), 100000)`.
  - Added status text behavior indicating when depth is capped at `100000`.
- Taxonomy-rank UI support:
  - Added `Strain` option to taxonomic level selectors in:
    - `modules/mod_ancom.R`
    - `modules/mod_biplot.R`
    - `modules/mod_heatmap.R`
    - `modules/mod_maaslin2.R`
    - `modules/mod_randomforest.R`
    - `modules/mod_sparcc.R`
    - `modules/mod_spieceasi.R`
- Strain parsing/labeling support:
  - Extended rank parsing defaults and prefix maps in:
    - `modules/mod_ancom.R`
    - `modules/mod_maaslin2.R`
    - `modules/mod_randomforest.R`
- Bar plot (`modules/mod_barplot.R`):
  - Added `Strain` tax level option and rank-prefix handling for `t__`.

## Tooling / Conversion
- MetaPhlAn conversion script (`scripts/generate_SM_format.py`):
  - Added robust clade-column detection (`clade_name`, `#clade_name`, etc.).
  - Defaulted agglomeration behavior to target rank with unassigned completion.
  - Default target rank set to `Strain`.
  - Updated output filenames:
    - `1_MASV_table.txt`
    - `2_taxonomy_table.txt`
    - `3_metadata_table.txt`
  - Updated output headers:
    - `#NAME` (MASV)
    - `#TAXONOMY` (taxonomy)
    - `SampleID` (metadata)
  - Removed `clade_name` column from taxonomy output.

## Documentation
- Updated manuals:
  - `docs/manual/alpha-diversity.md`
  - `docs/manual/data-loading.md`

## Citation
- Added `randomForest` package citation in:
  - `modules/mod_citation.R`
