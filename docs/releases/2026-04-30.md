# Release 2026-04-30

## Summary
- Increased file upload limit for larger input datasets.
- Refined Taxa Comparison regression workflow labels and defaults.
- Fixed continuous regression x-axis rendering and aligned taxa filtering behavior with regression mode.
- Added Jensen-Shannon distance metric option in Beta Diversity.

## Changed
- App configuration (`app.R`):
  - Added `options(shiny.maxRequestSize = 100 * 1024^2)` to set upload limit to 100MB.
- Taxa comparison (`modules/mod_taxa_comparison.R`):
  - In scatter + regression + continuous group mode, x-axis now renders as numeric continuous scale instead of categorical labels.
  - Updated UI text:
    - `Show Trend Line` -> `Show Regression Line`
    - `Trend Line Method` -> `Regression method`
  - Regression method choices reordered to list `Linear regression (lm)` first.
  - Regression method default restored to `Spearman`.
  - Legend text now adds regression-mode guidance (active method and pairwise bar behavior) when regression mode is enabled.
  - Taxa list filtering (`Show only p-value < 0.05`) now recalculates p-values using regression settings when regression mode is enabled.
- Manual (`docs/manual/taxa-profiles.md`):
  - Updated section naming to regression terminology.
  - Added explicit filtering calculation rule for regression mode vs non-regression mode.
- Beta diversity (`modules/mod_beta.R`):
  - Added `Jensen-Shannon` to `Distance metric` options and wired distance calculation with `method = "jsd"`.
  - Added `Jensen-Shannon` distance label in ordination/permanova reporting.
- Manual (`docs/manual/beta-diversity.md`):
  - Added `Jensen-Shannon` guidance to the distance-method section.

## Notes
- This release aligns on-screen labels, filtering logic, and documentation for regression-driven exploration in Taxa Comparison.
