# Release 2026-06-08

## Summary
- Standardized PNG download behavior across multiple analysis modules with 300 DPI export and more consistent plot-dimension handling.
- Improved SPIEC-EASI comparison outputs, including minimum edge-weight filtering and table layout spacing.
- Refined beta diversity EnvFit factor labeling and updated the workflow overview Taxa Profiles icon.

## Changed
- PNG exports:
  - Updated beta diversity PCoA/NMDS downloads to use user-selected plot dimensions with 300 DPI export.
  - Updated association biplot PNG download with explicit content type, run guards, payload validation, fallback dimensions, and 300 DPI export.
  - Updated heatmap PNG export to use inch-based dimensions with 300 DPI output.
  - Updated Random Forest SHAP and feature-importance plot downloads to export at 300 DPI while preserving configured plot dimensions.
  - Updated SPIEC-EASI network and comparison-network plot downloads to use 300 DPI output.
- SPIEC-EASI (`modules/mod_SpiecEasi.R`):
  - Applied the active `Minimum edge weight` threshold to differential edge comparison and comparison-network outputs.
  - Added table panel spacing around full edge, differential edge, and hub tables.
  - Added a network-run guard before building the reactive network result.
  - Adjusted differential edge table pagination for a more compact display.
- Beta diversity (`modules/mod_beta.R`):
  - Improved EnvFit factor-variable matching for factor levels with colon, dot, or prefix-style labels.
  - Cleaned factor-level labels using the resolved variable name.
  - Updated factor vector labels to display variable and level information.
- ANCOM-BC2 and MaAsLin2 (`modules/mod_ancom.R`, `modules/mod_maaslin2.R`):
  - Increased ranked bar-plot title font size relative to the selected base font size.
- Workflow overview (`www/workflow-overview.svg`):
  - Replaced the Taxa Profiles icon with a grouped stacked-bar composition graphic.
