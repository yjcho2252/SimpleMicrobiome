# Release 2026-06-09

## Summary
- Expanded alpha-diversity plot types, improved beta-diversity overlays, refined taxa filtering, and tightened file-load error handling.
- Updated manuals to match current filtering and statistical display behavior.

## Changed
- Alpha diversity (`modules/mod_alpha.R`):
  - Added `Violin plot` and `Jitter plot` options.
  - Moved download buttons into the plot area and removed the duplicate top-level buttons.
  - Updated legend text so bar-plot error bars are identified as standard error (SE).
  - Kept `Kruskal-Wallis` as an overall p-value display without pairwise brackets.
- Beta diversity (`modules/mod_beta.R`):
  - Moved PCoA/NMDS download buttons into their respective tabs.
  - Added optional centroid markers and sample-to-centroid line overlays.
  - Matched centroid/line colors to the group color palette.
  - Left centroid display disabled by default.
- Taxa Comparison (`modules/mod_taxa_comparison.R`):
  - Reverted the secondary-group facet layout to `facet_grid` after nested-facet testing.
- Taxa Bar plot (`modules/mod_barplot.R`):
  - Added an abundance-threshold display mode alongside top-N taxa selection.
- File load (`modules/mod_fileload.R`):
  - Added file-specific error messages for OTU, taxonomy, and metadata upload failures.
  - Kept the loader in an error state until the user re-uploads corrected files.
- Manuals:
  - Updated the ANCOM-BC2, MaAsLin2, and taxa-profiles manuals to reflect current filtering and plotting behavior.
