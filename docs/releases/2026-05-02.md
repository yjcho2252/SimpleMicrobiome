# Release 2026-05-02

## Summary
- Updated grouping behavior in Taxa Bar plot to allow `SampleID` selection while keeping non-`SampleID` defaults when available.
- Updated Taxa Comparison repeated-measures behavior for within-subject analysis with 3+ groups.
- Improved Beta Diversity legend wording for distance-metric readability.
- Clarified preprocessing sample-count wording.

## Changed
- Taxa bar plot (`modules/mod_barplot.R`):
  - Removed explicit exclusion of `SampleID` from primary/secondary group selectors.
  - Kept default preference for non-`SampleID` grouping variables when available.
- Taxa comparison (`modules/mod_taxa_comparison.R`):
  - In within-subject pairing mode with 3+ groups, overall test now uses `Friedman test` (repeated measures).
  - Updated status text to report repeated-measures overall testing accordingly.
- Beta diversity (`modules/mod_beta.R`):
  - Updated figure legend sentence to display non-Aitchison metrics as `<metric> distance` (for example, `Jensen-Shannon distance`).
- Preprocessing (`modules/mod_preprocessing.R`):
  - Updated selection summary text to `Total: N samples` and `Selected: M samples`.
- Manual documentation:
  - Updated `docs/manual/taxa-profiles.md` with current `SampleID` selector behavior in Taxa Bar plot grouping controls.
  - Updated `docs/manual/preprocessing.md` with current sample-count summary wording.

## Notes
- This release aligns UI wording and statistical behavior for clearer interpretation in paired/repeated workflows and grouping controls.
