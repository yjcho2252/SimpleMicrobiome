# Release 2026-05-03

## Summary
- Added `CCA` as an analysis method option in Association Biplot alongside existing `dbRDA`.
- Added convex hull overlays (`chull`) for group visualization in Beta Diversity ordination plots.
- Updated ellipse display to 80% confidence regions for clearer group spread interpretation.
- Updated manuals for Association Biplot and Beta Diversity to match current UI and behavior.

## Changed
- Association Biplot (`modules/mod_biplot.R`):
  - Added `Analysis method` selector with `dbRDA (capscale)` and `CCA`.
  - Split model fitting path by method:
    - `dbRDA`: distance-based (`vegdist` + `capscale`) with PERMANOVA outputs.
    - `CCA`: direct constrained ordination (`cca`) without distance-metric application.
  - Updated plot title/axis labels/status text dynamically by method (`CAP1/2` vs `CCA1/2`).
  - Updated legend text to state that distance metric is not used for CCA.
  - Added UI behavior to disable distance-metric input when `CCA` is selected.
- Beta Diversity (`modules/mod_beta.R`):
  - Added `Show Group Hulls` option and rendered group-wise convex hull polygons using `chull`.
  - Ensured hull fills follow primary-group custom colors consistently.
  - Set group ellipse level to `0.80` for both PCoA and NMDS.
  - Updated figure legend text to describe 80% ellipses and hull overlays when enabled.
- Manual documentation:
  - Updated `docs/manual/association-biplot.md` with analysis-method and distance-metric behavior.
  - Updated `docs/manual/beta-diversity.md` with group hull usage and interpretation cautions.

## Notes
- Hull overlays are descriptive boundaries and should not be interpreted as statistical tests.
