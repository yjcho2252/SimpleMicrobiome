# Release 2026-06-19

## Summary
- Pinned `CVXR` for ANCOM-BC2 compatibility, forced `ANCOMBC` reinstall after the pin, and aligned the installer with the current module dependency set.
- Moved prevalence filtering in ANCOM-BC2 and MaAsLin2 to occur after fixed/random effect terms are prepared.
- Updated beta ordination sidebar ordering and refreshed the branded icon assets.

  - Added `circlize` to the CRAN install list and kept the dependency list aligned with the current app and modules.
- ANCOM-BC2 (`modules/mod_ancom.R`):
  - Moved prevalence filtering to run after fixed and random effect terms are prepared.
  - Updated the result/status text so the filtering order is described accurately.
- MaAsLin2 (`modules/mod_maaslin2.R`):
  - Moved prevalence filtering to run after fixed and random effect terms are prepared.
  - Updated the result/status text so the filtering order is described accurately.
- Beta diversity (`modules/mod_beta.R`):

- Branding:

  - Updated the README branding image to use the new icon asset.
