# Release 2026-06-30

## Summary
- Fixed ANCOM-BC2 bar-plot axis handling so positive-only LFC results render bars correctly.
- Prevented SparCC from starting network estimation before the user clicks `Run SparCC`.
- Improved SparCC table-tab spacing so DataTables no longer overlap the legend/status area.
- Added TSV downloads for SparCC and SpiecEasi `Differential Edges` and `Hub Table`.
- Restored reliable browser tab favicon rendering by linking directly to the PNG icon asset.
- Updated manuals to match the current module UI and server behavior.

## Fixed
- ANCOM-BC2 (`modules/mod_ancom.R`):
  - Kept the ranked bar plot axis anchored at zero so bars remain visible when all selected LFC values are positive.
  - Filtered bar-plot input to finite LFC values before ranking and axis calculation.
- SparCC (`modules/mod_sparcc.R`):
  - Changed the network run trigger so selecting `Compare two groups` does not start NetCoMi/SparCC estimation before the run button is clicked.
  - Added table-panel spacing for `Table`, `Differential Edges`, and `Hub Table` to prevent DataTables output from overlapping the legend.
  - Added TSV download buttons for `Differential Edges` and `Hub Table`.
  - Fixed `Comparison Network` PNG downloads by explicitly rendering the ggplot object before closing the graphics device.
- SpiecEasi (`modules/mod_SpiecEasi.R`):
  - Added TSV download buttons for `Differential Edges` and `Hub Table`.
- App shell (`app.R`):
  - Switched the favicon link from the SVG wrapper to the 64px PNG icon asset for browser compatibility.
  - Aligned `www/favicon.svg` with the 260617 PNG icon asset for consistency.

## Changed
- Manuals:
  - Rewrote the ANCOM-BC2 manual to clarify reference-level direction, `diff = TRUE`/q-value behavior, bar-plot top-10 logic, trend testing, covariates, and random effects.
  - Rewrote the MaAsLin2 manual to document automatic normalization/transform settings by analysis method (`LM = TSS + LOG`, `ZINB = NONE + NONE`).
  - Rewrote the SparCC manual to remove unsupported p-value threshold language and document minimum edge-weight filtering plus two-group comparison behavior.
  - Rewrote the SpiecEasi manual to describe current analysis modes, edge filtering, comparison outputs, and hub summaries.
  - Rewrote the Correlation Heatmap manual around current group type, association mode, value scale, transform, correlation method, and FDR masking controls.
  - Updated the Taxa Profiles manual to distinguish q-value taxa filtering from p-value bracket filtering.
  - Updated the Random Forest manual to list the current result tabs and Macro-F1 metric.
