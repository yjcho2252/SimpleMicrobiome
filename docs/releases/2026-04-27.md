# Release Notes - 2026-04-27

## Release Metadata
- Branch: `main`
- Scope: Workflow figure refresh, subgroup UI refinement, plotting-scale alignment, and license metadata update

## Summary
This release refreshes the app workflow overview graphic and improves consistency across major analysis modules. It introduces clearer subgroup selection controls, aligns Alpha Diversity bar-plot baseline behavior with Taxa Comparison, updates Heatmap plot-dimension labeling, and revises the MIT license copyright holder to Kangwon National University.

## Changes

### 1) Workflow overview refresh
- Replaced the existing module-only workflow graphic with a full app workflow:
  - Data Loading
  - Preprocessing
  - Analysis Modules
  - Outputs
- Added module branches for:
  - Taxa Profiles
  - Diversity
  - Differential analysis
  - Network analysis
  - Association analysis
- File:
  - `www/workflow-overview.svg`

### 2) Subgroup selection UI refinement
- Added `Select subgroup` toggles to keep primary/secondary grouping controls hidden unless needed.
- Reset subgroup-specific inputs back to `All` / `None` when subgroup mode is disabled.
- Renumbered visible controls so the sidebar flow remains sequential when subgroup controls are hidden.
- Improved subgroup-aware metadata column resolution and selection updates.
- Files:
  - `modules/mod_ancom.R`
  - `modules/mod_maaslin2.R`
  - `modules/mod_randomforest.R`
  - `modules/mod_sparcc.R`
  - `modules/mod_SpiecEasi.R`
  - `modules/mod_heatmap.R`

### 3) Alpha Diversity bar-plot scale behavior
- Updated Alpha Diversity bar plots to follow the same baseline approach used in Taxa Comparison.
- Bar plots now use facet-aware `scale_min` values rather than forcing bars to start at zero.
- Applied this behavior to both standard bars and `ggpattern` bars when supported.
- File:
  - `modules/mod_alpha.R`

### 4) Heatmap and preprocessing UI polish
- Heatmap:
  - Renamed `Cell Size` to `Plot Dimensions`.
  - Updated the section icon.
  - Moved transform/correlation controls into `Advanced options`.
- Preprocessing:
  - Added a small visual gap between `Select All` and `Unselect All`.
- Files:
  - `modules/mod_heatmap.R`
  - `modules/mod_preprocessing.R`

### 5) Additional UI and metadata updates
- Refined compact sidebar text and placeholder styling in association biplot/file loading flows.
- Updated the MIT license copyright holder to `Kangwon National University`.
- Files:
  - `modules/mod_biplot.R`
  - `modules/mod_fileload.R`
  - `LICENSE`

## Notes
- No migration steps are required.
- Detailed changes are also recorded in `CHANGELOG.md`.
